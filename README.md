# Airbnb-NYC-Data-Cleansing-Structuring-Visualization

## Project Overview

This project focuses on analyzing Airbnb listing data from New York City using Microsoft Excel and Power BI. The primary objective was to clean raw data, perform exploratory analysis, identify business insights, and build an interactive dashboard for visual storytelling.

The project demonstrates practical data analytics skills including data cleaning, descriptive analysis, KPI tracking, dashboard development, and business insight generation.

---

## Tools & Technologies Used

- Microsoft Excel
- Power BI
- Pivot Tables
- Data Cleaning
- Data Visualization
- Dashboard Design

---

## Dataset Information

The dataset contains 10,000 Airbnb listings across New York City with 22 different attributes related to hosts, pricing, room types, reviews, availability, and location details.

The data was used to analyze pricing behavior, customer preferences, room distributions, and borough-level trends within NYC's Airbnb market.

---

# Dataset Columns Description

| Column Name | Description |
|---|---|
| `id` | Unique identifier for each Airbnb listing |
| `NAME` | Title/name of the Airbnb listing |
| `host id` | Unique identifier of the host |
| `host_identity_verified` | Indicates whether the host identity is verified |
| `host name` | Name of the Airbnb host |
| `neighbourhood group` | NYC borough where the property is located (Brooklyn, Manhattan, Queens, etc.) |
| `neighbourhood` | Specific neighborhood within the borough |
| `lat` | Latitude coordinate of the property |
| `long` | Longitude coordinate of the property |
| `instant_bookable` | Indicates whether the property can be instantly booked |
| `cancellation_policy` | Cancellation policy offered by the host |
| `room type` | Type of room listed (Entire home, private room, shared room, etc.) |
| `Construction year` | Year in which the property was constructed |
| `price` | Listing price of the property |
| `service fee` | Airbnb service fee charged |
| `minimum nights` | Minimum number of nights required for booking |
| `number of reviews` | Total number of customer reviews |
| `last review` | Date of the most recent review |
| `reviews per month` | Average monthly review count |
| `review rate number` | Average customer rating score |
| `calculated host listings count` | Total listings managed by the same host |
| `availability 365` | Number of days the listing is available in a year |

# Data Cleaning Process

The following preprocessing steps were performed in Excel:

- Removed duplicate records
- Handled missing/null values
- Standardized inconsistent text formatting
- Cleaned pricing and service fee columns
- Converted data types where necessary
- Validated numerical columns
- Checked for inconsistent neighborhood naming

# Key Insights

- Manhattan listings showed the highest average pricing among all boroughs.
- Entire homes/apartments were significantly more expensive than private rooms.
- Certain neighborhoods had high listing availability but lower pricing trends.
- Review activity was concentrated in popular tourist-heavy areas.
- Pricing distribution contained several premium outlier listings.
- Verified hosts generally maintained higher review activity.
